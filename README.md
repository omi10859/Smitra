# Shram Mitra Landing Page

This is the static landing page for the Shram Mitra website, built with Next.js and shadcn/ui.

## Build Contents

Check the `build` branch for the latest build contents and deployment-ready files.

## Getting Started

This project uses Bun as the package manager, but you can also use npm.

### Installation

```bash
# With Bun
bun install

# With npm
npm install
```

### Development

```bash
# With Bun
bun run dev

# With npm
npm run dev
```

### Build

```bash
# With Bun
bun run build

# With npm
npm run build
```
